```sql
-- 1. 실행 전 예상 플랜
EXPLAIN PLAN FOR
SELECT * FROM so_seg_usg WHERE inst_no = 1;

SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```

```sql
-- 2. 실행 후 실제 플랜 (sqlplus)
SET AUTOTRACE ON
SELECT * FROM so_seg_usg WHERE inst_no = 1;
```

```sql
-- 3. 실행 후 실제 플랜 (SQL_ID로)
SELECT /*+ GATHER_PLAN_STATISTICS */ *
FROM so_seg_usg WHERE inst_no = 1;

SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY_CURSOR(
    (SELECT prev_sql_id FROM v$session WHERE sid = SYS_CONTEXT('USERENV', 'SID')),
    0,
    'ALLSTATS LAST'
));
```

| 방법 | 실제통계 | 간편함 | 도구 |
|------|---------|--------|------|
| EXPLAIN PLAN | X | O | 모두 |
| AUTOTRACE | O | O | sqlplus |
| GATHER_PLAN_STATISTICS | O | X | 모두 |
